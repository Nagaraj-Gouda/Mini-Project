# Student Management System (MERN) - Basic CRUD
This is a minimal MERN stack project (MongoDB, Express, React, Node) implementing **Basic CRUD for Students**.

## Structure
- backend/ - Express API
- frontend/ - React app (minimal, uses fetch to talk to backend)

## Notes
- Backend expects a MongoDB URI in `backend/.env` as `MONGO_URI`.
- To run backend:
  ```
  cd backend
  npm install
  npm run dev
  ```
- To run frontend:
  ```
  cd frontend
  npm install
  npm start
  ```
